# User Stories - Airbnb Clone

This document translates our system use cases into agile user stories.

1. **User Registration:** As a potential user, I want to create an account so that I can manage my bookings and listings.
2. **Property Search:** As a guest, I want to search for properties by destination and date so that I can find a place that fits my trip.
3. **Booking Process:** As a guest, I want to book a property and receive instant confirmation so that my travel plans are secured.
4. **Host Listing:** As a host, I want to list my property with photos and pricing so that I can attract guests.
5. **Secure Payments:** As a guest, I want to pay for my stay via a secure checkout process so that my financial information is protected.
6. **Reviews:** As a guest, I want to leave a review after my stay so that I can share my feedback with the community.
