# System Process Flowcharts

## Overview
This folder contains the flowchart for the **Property Booking Process**. This diagram details the logical sequence and decision points required to complete a reservation.

## Process Workflow
1. **Initiate:** Guest selects a property and dates.
2. **Availability Check:** System verifies if the property is free.
3. **Decision:** If unavailable, return to search; if available, proceed to payment.
4. **Payment Processing:** Interface with the payment gateway.
5. **Confirmation:** If payment succeeds, save booking and notify the guest.
