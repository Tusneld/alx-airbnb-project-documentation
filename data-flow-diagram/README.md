# Data Flow Diagram (DFD)

## Overview
This directory contains the Data Flow Diagram (DFD) for the Airbnb Clone backend. It illustrates the movement of data between external entities, system processes, and internal data stores.

## Diagram Components
- **Entities:** Guest, Host, Payment Gateway.
- **Processes:** User Authentication, Property Management, Booking Logic, Payment Processing.
- **Data Stores:** Users Table, Properties Table, Bookings Table.

## Data Flows
1. **User Input:** Login credentials and search queries sent to the system.
2. **System Output:** Property results and booking confirmations sent to the user.
3. **Database Transactions:** Booking records stored and retrieved from the database.
